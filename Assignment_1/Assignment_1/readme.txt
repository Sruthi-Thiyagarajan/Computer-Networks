Please follow below steps for Server Program Compilation: 

	1. Type the following command to compile server program       			- 		gcc -o server server.c 
	2. Type following command to start the server                 			-		./server

Please follow below steps for the Client Program Compilation:

	1. Type the following command to compile client program         		- 		gcc -o client client.c
	2. Type following command to start the client and start sending packet  - 		./client

Note : Always start the server first and then client.
